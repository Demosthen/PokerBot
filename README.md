# PokerBot

We programmed a Rethink Robotics Baxter robot to play a simple card game using its camera and a suction cup gripper! A more thorough report can be found here: https://sites.google.com/berkeley.edu/eecs106afinalreport/results?authuser=0 along with a demo video: https://youtu.be/AaV-ndfVGuw
